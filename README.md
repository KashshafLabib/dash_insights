# TakaPay Pulse

Turn a raw multilingual social-media feed about TakaPay into insights a brand manager can act on.

> Full documentation, architecture, and the live demo link land here as the build progresses.

## Structure

- `data/` — the source dataset (660 posts, Bangla / Banglish / English)
- `analysis/` — Python: exploratory analysis notebook and the offline audit pipeline
- `web/` — Next.js (TypeScript) dashboard and API
